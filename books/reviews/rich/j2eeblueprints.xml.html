Designing Enterprise Applications with the Java 2 Platform: Enterprise Edition

by Nicholas Kassem

( Amazon )

The J2EE "blueprints" book. Good but dull.

This is a collection of chapters by various authors describing the J2EE architecture and the best practices for using it. It is a book I will never finish reading because I just can't stay awake for even a single chapter. Very high level, full of enterprise buzz words, probably very good for getting an overview in some detail (if you see what I mean).

If you need this book (which, they were giving away at JavaOne 2001) try it on-line for free. For a more hands-on text, try something like Professional Java Server Programming J2EE, 1.3 Edition.

My rating: Good, but dull

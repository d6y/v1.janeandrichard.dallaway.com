Database Programming with JDBC and Java, Second Edition

by George Reese

( Amazon )

An OK guide to JDBC.

I have the first edition of this book, but the change in cover and layout fooled me into checking it out on line at O'Reilly's subscription site. I was looking for information on some of the features of JDBC 2. This second edition was printed only as JDBC 2 was released, so the information is short and slightly speculative. Look out for a third edition.

Or don't. The text is long winded. If you need a slow introduction, consider this text. Otherwise, look for a single chapter in another book, such as Professional Java Server Programming J2EE, 1.3 Edition.

My rating: Nothing special

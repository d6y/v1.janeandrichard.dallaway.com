Java and XSLT

by Eric M. Burke

( Amazon )

The best text covering the use of XSLT from Java.

For use of Java with the JAXP API and Xalan 2, and the use of XSL-T in servlets, I've not seen a better text. Clear, good examples.

I've not spent much time with the earlier chapters on the basics of XSL-T: for that I use the XML Nutshell or the comprehensive XSLT Programmer's Reference.

My rating: Recommended

Apache: The Definitive Guide

by Ben Laurie, Peter Laurie

( Amazon )

Don't bother: use the Apache web site instead.

I purchased the first edition of this book, and I didn't think much of it then. I've just had another look on-line at the O'Reilly Safari on-line subscription web site, and I still don't like it.

My problem with it is that it doesn't help me solve problems. When I look something up in this book, the "real" information ends up being in the same format as the Apache on-line documentation. There's not a lot added that I found of use.

My rating: Useless

Learning Python

by Mark Lutz, David Ascher

( Amazon )

I've heard good things about Python, so I thought I'd take a look.

This is a throw away book: you use it to get the basics of the language, and then move on. It gives a few examples of some of the interesting libraries available, but not enough to really get you building applications. For example, I'm interested in the regular expressions and GUI libraries, but these topics are given the briefest coverage. And that's OK -- this one book does enough giving you an understanding of language. But having got that, I don't think I'll be looking at this text again.

My rating: Good
